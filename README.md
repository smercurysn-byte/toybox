# 🧸 나만의 토이박스 (My Toybox)

바닐라 HTML/CSS/JavaScript로 만든 캔버스 기반 미니 장난감 모음. 빌드 도구 없이 `index.html`을 브라우저로 열기만 하면 동작합니다.

## 장난감

| 탭 | 설명 |
|---|---|
| **요요** | 색상 커스터마이징 + 슬립/전문 기술 8종 (숫자키 1~8) |
| **핑거보드** | 데크 튜닝 + 공중 플립·매뉴얼·그라인드 등 트릭 (키보드) |
| **키캡** | 기계식 키보드 키캡 색칠 + 실제 타건 애니메이션/타건음 |
| **🎲 큐브** | 루빅스 큐브 전개도, 랜덤 스크램블, 스페이스바 스톱워치, 패턴 도감 |
| **🏎️ 미니카** | 가상 차고지 등록 + 랜덤 트랙 레이아웃 설계 & 주행 애니메이션 |
| **🔊 피젯** | 피젯 스피너·팝잇·패드 ASMR 사운드 믹서 + 10초 연타 모드 |
| **🖊️ 펜돌리기** | 펜 돌리기 기술 콤보 슬롯머신 (난이도 필터) |
| **🥃 테이스팅** | 위스키/커피 맛을 5각 레이더로 기록하는 바 캐비닛 + 오늘의 추천 |
| **🪵 데스크** | 데스크테리어 소품 배치 + 반려식물 광합성/급수 타이머 |
| **🧩 퍼즐** | 절차적 명화 직소 퍼즐 + 빗소리·모닥불·턴테이블 백색소음 |
| **🖋️ 잉크** | 만년필 잉크 방울 차트 + 오늘의 필사 문장 |

## 구조

```
index.html      마크업 (탭 6개 + 각 스테이지)
styles.css      스타일
js/
  core.js       공유 유틸·저장소·오디오·캔버스·탭 (window.TB)
  yoyo.js  board.js  keycap.js  cube.js  cars.js  fidget.js
```

- 일반 `<script>`로 로드되어 `file://` 더블클릭에서도 동작합니다. `core.js`가 먼저 `window.TB`(공유 유틸)를 만들고 각 장난감 모듈이 이를 사용합니다.
- 설정·기록은 브라우저 `localStorage`에 저장됩니다.

## 실행 / 배포

- **로컬**: `index.html` 더블클릭
- **배포**: 정적 파일이라 GitHub Pages 등 어떤 정적 호스팅에도 그대로 올릴 수 있습니다.
